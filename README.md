# limpiatexto

limpiador de formato de texto. pega texto con formato (html, word, etc.) y cópialo limpio.

## funcionalidades

- pega texto con cualquier formato y lo convierte a texto plano
- toggle minúsculas (Alt+L)
- toggle eliminar saltos de línea (Alt+N)
- botón copiar texto limpio al portapapeles
- estado de toggles guardado en localStorage

## tecnologías

- html5, css3, javascript vanilla
- sin frameworks ni librerías externas
- procesamiento 100% en el navegador

## uso

abre `index.html` en un navegador o despliega en cualquier hosting estático.
